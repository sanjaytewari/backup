Queries of MongoDb (To run in MongoShell): 
-> copy connection string from MongoDb compass and paste it on MongoShell
-> To see existing databases : Show dbs;
-> To create a database: show <database name> (example-> show Student)
-> To use a database: show <database name>
-> To see existing collections : show collections;
-> To cretae collection: db.createCollection("<database name>") (example-> db.createCollection("StudentInfo"))
-> Insert single document in a collection with example: db.StudentInfo.insertOne({Name: "Tushar", RollNo: "420"});
-> Insert multiple document in a collection with example: db.StudentInfo.insertMany([{Name: "Rohit", RollNo: "320"},{Name: "Vasu", RollNo: "520"}]);
-> To get all data of collection with example : db.StudentInfo.find();
-> To find all data except field assigned as 0 with example : db.StudentInfo.find({},{_id:0});
-> To update a collection document with example :
	 Student> db.StudentInfo.update({RollNo:"320"}, {$set :{Name :"ROHIT",Age:"22"}},{upsert: true});
-> To find data in collection excluding autogenerated _id column where Name is either Vasu or Tushar : 
	 Student> db.StudentInfo.find({Name:{$in: ["Tushar","Vasu"]}},{_id:0});
